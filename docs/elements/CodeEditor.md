# `<CodeEditor>`

CodeEdtidor element wraps the Juce CodeEditorComponent.

## Attributes

| Attribute   | Description                                 |
|:------------|:--------------------------------------------|
| `text`      | Text being edited                           |
| `readonly`  | `true` for read-only text field             |
| `onchange`  | Text changed event handler                  |
