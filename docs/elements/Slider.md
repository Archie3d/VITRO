# `<Slider>`

This element corresponds to Juce Slider.

# Attributes

| Attribute  | Description                                      |
|:-----------|:-------------------------------------------------|
| `value`    | Slider value                                     |
| `min`      | Value range minimum                              |
| `max`      | Value range maximum                              |
| `step`     | Value increment step                             |
| `decimals` | Number of digits to show after the decimal point |

# Style properties
